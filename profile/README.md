## 🔥 서비스 소개

> 슬기로운 긱사생활은 기숙사생들이 배달, 공구, 택시 등 생활 목적에 맞는 파티를 모집하고, 모집 완료 후 자동 생성되는 채팅방에서 실시간으로 소통할 수 있는 기숙사 커뮤니티 플랫폼입니다.

기숙사 생활에서는 배달을 함께 시키거나, 택시를 같이 타거나, 필요한 물건을 공동 구매하는 상황이 자주 발생합니다. 하지만 기존 학교 커뮤니티에서는 관련 정보가 여러 게시글에 흩어져 있어 원하는 정보를 찾기 어렵고, 모집이 완료된 이후에는 별도의 오픈채팅방을 만들어야 하는 불편함이 있었습니다.

저희는 이러한 문제를 해결하기 위해 기숙사생들이 목적별로 파티를 모집하고, 모집 인원이 충족되면 채팅방을 통해 자연스럽게 소통할 수 있는 서비스를 기획하게 되었습니다.

<br/>

## 📌 주요 기능

| 기능 | 설명 |
| --- | --- |
| 👤 회원가입 | 사용자는 이메일, 비밀번호, 이름, 호실, 기숙사 정보를 입력해 가입할 수 있습니다. |
| 🔐 로그인 | 가입한 이메일과 비밀번호를 통해 자신의 계정으로 로그인할 수 있습니다. |
| 🏠 메인페이지 | 최근 모집 중인 파티와 배달, 공구, 택시, 자유게시판으로 이동할 수 있는 카테고리를 제공합니다. |
| ✍️ 글 작성 | 사용자는 제목, 내용, 모집 인원, 약속 시간, 장소를 입력해 파티 모집 게시글을 작성할 수 있습니다. |
| 📄 상세 페이지 | 게시글 정보와 파티 모집 정보를 확인하고, 댓글 작성 및 파티 참여가 가능합니다. |
| 🤝 파티 참여 | 사용자가 파티에 참여하면 참여자 정보가 저장되고, 모집 인원이 충족되면 파티 상태가 자동으로 변경됩니다. |
| 💬 실시간 채팅 | 파티 모집 완료 후 자동 생성된 채팅방에서 참여자들이 실시간으로 소통할 수 있습니다. |

### 시스템 흐름

```text
회원가입 / 로그인
        ↓
메인페이지 접근
        ↓
카테고리별 게시판 조회
        ↓
파티 모집 글 작성
        ↓
게시글 + 파티 정보 저장
        ↓
사용자 파티 참여
        ↓
모집 인원 충족 시 파티 상태 변경
        ↓
채팅방 자동 생성 및 실시간 소통
